# 🧮 Simple Calculator App

A sleek, responsive calculator built with HTML, CSS, and vanilla JavaScript. This project provides a clean interface for basic arithmetic operations.

## ✨ Features

- **🔢 Basic Operations**: Perform addition, subtraction, multiplication, and division.
- **🔙 Backspace Support**: Easily correct mistakes by deleting the last character.
- **🧹 Clear Function**: Reset the calculator with a single click.
- **📱 Responsive Design**: Fully optimized for both desktop and mobile devices.
- **🎨 Modern UI**: Features a clean, centered layout with interactive buttons and custom fonts.

## 🛠️ Technologies Used

- **HTML5**: Structured with semantic elements.
- **CSS3**: Modern styling, including flexbox for layout and custom Google Fonts (Inter).
- **JavaScript (ES6)**: Pure vanilla JS for high performance and logic handling.
- **Font Awesome**: Used for intuitive icons (backspace).

## 📂 Project Structure

- `index.html`: Main application interface.
- `index.css`: Stylesheet for the calculator design and layout.
- `index.js`: Functional logic for mathematical computations.
- `logo-1.webp`: Project branding and preview image.

## 🚀 Getting Started

1.  Clone this repository to your local system.
2.  Open `index.html` in any modern web browser.
3.  Start calculating!
